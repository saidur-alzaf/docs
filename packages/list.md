## 🤝 Contributing

নতুন package যোগ করতে চাইলে:

1. এই repo fork করুন
2. নতুন package সঠিক category-তে যোগ করুন
3. Pull Request পাঠান

**Format অনুসরণ করুন:**
```md
| [package-name](link) | ![npm](badge-url) | Short description | `npm i package-name` |
```
